## Run Locally

# Clone the project

```bash
  git clone https://github.com/Kunal-Ch-Das-Official/official-portfolio.git
```

# Go to the project directory

```bash
  cd official-portfolio
```

# Install dependencies

```bash
  npm install
```

# Start development environment

```bash
  npm run dev
```

# Start prouction environment

```bash
  npm run build
```
npm start
